Troubleshooting

App Support Page

Common Issues

If you're experiencing issues with Scrollless, please try the following steps:

App Not Working

Make sure you have granted all necessary permissions to the app, including:

  • Usage Access permission
  • Notification access (if applicable)
  • Display over other apps permission

Timers Not Triggering

Ensure that:

  • The app is not being battery optimized (disable battery optimization for Scrollless)
  • Background restrictions are turned off for the app
  • You have set up timers correctly in the app settings

Break Screen Not Appearing

If the break screen doesn't appear when a timer expires:

  • Check that "Display over other apps" permission is enabled
  • Make sure the app is running in the background
  • Verify that you haven't accidentally dismissed or snoozed the break

Still Need Help?

If you're still experiencing issues, please contact us at:

scrollless.app@gmail.com

  • Your device model and Android version
  • The specific issue you're facing
  • Steps you've already tried
